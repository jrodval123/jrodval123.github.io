Project Description

We are exploring and improving an open-source object/person tracking and an algorithm. Our robot will be able to follow the object/persons while moving through a room.

1

The way it works

The implemented code works by using the incorporated 3D sensor to create a box of point cloud and fixating on the object/person that is in the centroid of the box generated.

2

Motivation

Spot, the Boston Dynamics robot
motivated us to pursue this project,
to have a better understanding on how
such technologies work
and how they can be applied to multiple situations.

3


Objectives

Implement and improve upon the turtlebot follower algorithm

Test turtlebot’s ability to follow an object/person

Test turtlebot’s ability to avoid collision

4

Goals

The robot should be able to follow the object/persons while moving through a room without losing track of it.

The robot needs to be able to keep a certain distance from the person/object that it is following.

The robot should be able to avoid collision with other objects as it moves around.

5

Current Status

Familiarize ourselves with ROS.

Got accustomed with using the Turtlebot and its basic activities.

Integrated the source code successfully

6

Complications

We encountered hardware issues such as a non-functioning 3D sensor which may have been due to faulty installation of ROS on the computer. After a meticulous revision of the filesystem on the computer, we saw that files like bashrc were missing some exports, and there was a directory missing as well. We solved this by switching computers, since we were instructed not to edit that aspect.
